# alcledlight.com static website

This is a GitHub Pages-ready static website for `alcledlight.com`.

## Upload method

1. Create a GitHub repository.
2. Upload all files in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Choose `main` and `/root`.
6. Add the custom domain: `alcledlight.com`.
7. Keep the included `CNAME` file.

## Main files

- `index.html` — homepage
- `products/milky-white-silicone-neon-flex/index.html` — product detail page
- `contact/index.html` — contact page
- `assets/css/site.css` — one main stylesheet
- `assets/js/site.js` — lightweight menu/year script
- `robots.txt` and `sitemap.xml` — basic SEO files
